JSON 文件格式说明

导入的 JSON 文件需符合以下格式。导出时会自动生成符合此格式的文件。

食谱格式示例
{ "recipes": [{ "name": "番茄炒蛋", "cuisine": "家常菜", "difficulty": "简单", "prep_time": 10, "cook_time": 15, "tags": ["家常", "快手菜"], "description": "酸甜可口", "steps": ["第一步...", "第二步..."], "ingredients": [{ "name": "鸡蛋", "amount": 2, "unit": "个" }] }] }
食材格式示例
{ "ingredients": [{ "name": "鸡胸肉", "category": "肉类", "nutrition_per_100g": { "calories": 133, "protein": 31.0, "fat": 3.6, "carbs": 0, "fiber": 0, "vitamin_a": 0, "vitamin_c": 0, "calcium": 5, "iron": 1.0 } }] }
注意事项：导入时同名食材或食谱不会重复创建。导入的 JSON 文件需包含完整的食材名称（必须与食材库中的名称一致），否则食材关联将失效。建议优先导出食材数据，再导出食谱数据。
